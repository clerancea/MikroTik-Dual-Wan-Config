# MikroTik Dual WAN Load Balancing & Failover

## Overview

This repository documents the MikroTik dual-WAN implementation used to provide:

- Internet redundancy
- WAN failover
- Load balancing
- Network resiliency

The deployment consists of:

- WAN1 (Static Public IP)
- WAN2 (DHCP Internet Connection)
- Internal LAN
- Source NAT
- PCC Load Balancing
- Automatic WAN failover
